# [Error SOP][150010018]Invalid handling time

| Error Code | Error Message |
|---|---|
| 150010018 | Invalid handling time |

- 

**Root cause:**

shipmentLimitDay parameter is in correct.

| Properties | Type | Required | Description |
|---|---|---|---|
| shipmentLimitDay | INTEGER | True | Indicates the time, in days, between when you receive an order for an item and when you can ship the item. The default production time is 1, 2.In some scenarios, longer production times are supported. |

- 

**From seller center perspective:**

The "Handling time" provided in the product must meet the requirements in Temu seller center.
