# Update IP Address or Support Site

**This document helps you update IP address and support site.**

*IP address is filled in "Compliance and security assessment" Question 5 and serves as the unique numeric identifier for sending API requests to TEMU.*

*Support site is filled in "Compliance and security assessment" Question 2 and related to the shipping information of orders.*

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/ab9a487e1bc17a058dad6fb49600c1ef)

---

## 

1.Update IP Address

#### 

1.1 Choose your online app and click "Edit app information".

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/ead5be8041486c00fdefd5311ce0c9e3)

#### 

1.2 Choose "Only IP address list and Support site".

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/fc0ba615425819c310442b7cd7d66462)

#### 

1.3 Fill in IP address in Question 5 and change the answer of Question3,4,6 at the same time if needed.

- 

*Question 3: select all countries/regions the servers for processing personal data located including IP address and self-constructed data centers.*

- 

*Question 4: select all cloud vendors used to process personal data including IP address and self-constructed data centers.*

- 

*Question 6: When cloud vendors are selected in Question 4 and Subprocessors list will show Data hosting automatically. You can check and recorrect the answer.*

**Note that:**

- 

**"Self-constructed data centers" can not be selected and shown in ****Subprocessors list>>Data hosting>>Provider Subprocessor Name**** because it does not belong to subprocessors. Therefore. the ****Subprocessors list>>Data hosting>>Provider Subprocessor Name**** is shown as same as Question 4, excluding "self-constructed data centers".**

- 

**The answer of ****Subprocessors list>>Data hosting>>Location**** is a subset of the answer to Question 3.**

> 

**For example:**

> 

The current IP address is from Google located in United States.

> 

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/76f32b69fd310ac09a501f2480244e01)

> 

> 

User Lindy wants to add a new IP address from Amazon located in France and self-constructed data centers located in Germany, which are different from the old one.

> 

step1: append "France" depending on IP address and "Germany" depending on self-constructed data centers in Question 3.

> 

step2: append "Amazon" depending on IP address and "self-constructed data centers" in Question 4.

> 

step3: check whether the answer of Subprocessors list>>Data hosting>>Provider Subprocessor Name is correct.

> 

step4: check whether the answer of Subprocessors list>>Data hosting>>Location is correct.

> 

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/61b004ecde455176a174ad34eae5e768)

#### 

1.4 Submit this form and wait for reviewing.

- 

App Management

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/a12a21839d341bf8d0e8fcaccfeb58ff)

- 

App management>>App details

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/41aa20bc5fd5595c897570bb306dc90a)

#### 

1.5 When your form is approved, the updated information will be shown in "Compliance and security assessment", "App details" and "My Protocol".

- 

"Compliance and security assessment" and "App details" shows updated IP address.

- 

"My Protocol" shows updated Subprocessors list.

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/c03616443c71e84fe9b1365238ba33e3)

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/4028e299fe274661d2f6e620788ce1a6)

## 

2.Update Support Site

#### 

2.1 Choose your online app and click "Edit app information".

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/dc563fd35a5e2f5a084b74df3f5d0ce6)

#### 

2.2 Choose "Only IP address list and Support site".

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/adf98b7941720fcd06b52cac79b6d957)

#### 

2.3 Select the countries/regions you offer services to merchants.

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/468deb2642ff9c1100ff8538db688abc)

#### 

2.4 Submit this form and wait for reviewing.

- 

App Management

![image](https://bstatic.kwcdn.com/open-outer/20ef5f3450/be917270dc3ff30acac231cbb31b8691)

- 

App management>>App details
