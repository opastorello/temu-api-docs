# [Error SOP][150010031]Shipping template not found

| Error Code | Error Message |
|---|---|
| 150010031 | Shipping template not found |

- 

**Potential Cause:**

1. 

Shipping template is not from the current shop in temu seller center.

1. 

Shipping template has been deleted by the seller 

- 

**Next Step**

Reterive the newest costTemplateId from [bg.freight.template.list.query](https://partner-eu.temu.com/documentation?menu_code=7289390cfd724be4a196f11ebe45a896&sub_menu_code=d2e6f9fce8304634a85c3b808c19e0cd).

*Please note sellers need to go to the seller center to create the shipping template page in order for this interface to return the corresponding data
