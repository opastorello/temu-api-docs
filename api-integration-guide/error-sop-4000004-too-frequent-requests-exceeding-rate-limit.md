# [Error SOP][4000004]too frequent requests, exceeding rate limit.

| Error Code | Error Message |
|---|---|
| 4000004 | too frequent requests, exceeding rate limit. |

- 

**Root Cause:**

Requests are too frequent and exceed the rate limit. Please refer to the QPS limitation guide.

ref: [https://partner-eu.temu.com/documentation?menu_code=85762c6ccc5a4dbc8c023ea5e10c6dc0&sub_menu_code=17dc3fac0b684986ba5b131a70a05f58](https://partner-eu.temu.com/documentation?menu_code=85762c6ccc5a4dbc8c023ea5e10c6dc0&sub_menu_code=17dc3fac0b684986ba5b131a70a05f58)

*Besides interface-level limits, appkey-level limits also apply. Recommended: under 200 QPS per appkey.
