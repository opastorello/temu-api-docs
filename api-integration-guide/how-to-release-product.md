# How to release product ?

# 

Overview

The Product API helps sellers manage their product catalog at scale. Using the Product API, a seller can create products.

The following is an explanation of the field values and API calls involved in product publishing. We hope this document will help you to publish products more efficiently.

The following are the interfaces involved in product release

[](https://partner.temu.com/documentation?menu_code=38e79b35d2cb463d85619c1c786dd303&sub_menu_code=5bb50d82c08740688068d960977f8d52)

| API name | Interface Overview |
|---|---|
| bg.local.goods.cats.get | Get the complete category of temu |
| bg.local.goods.template.get | Obtain attribute/variant data corresponding to temu category |
| bg.local.goods.spec.id.get | Used to generate custom variant specification IDs |
| bg.local.goods.size.element.get | Whether to fill in the size chart and requirements for obtaining categories |
| bg.local.goods.image.upload | Upload product images |
| Upload videos, files, and imagesReference Documentsbg.local.goods.gallery.signature.get | Upload product videos, files, and images |
| bg.local.goods.compliance.property.check | Verify whether the product attributes comply with the site's sales rules |
| bg.local.goods.tax.code.get | Obtain the product tax code |
| bg.local.goods.out.sn.checkbg.local.goods.sku.out.sn.check | Verify if the external product code and SKU code are duplicated |
| bg.freight.template.list.query | Get the seller's set product shipping template |
| temu.local.goods.brand.trademark.V2.get | Obtain the brand information registered by the seller |
| temu.local.goods.sku.net.content.unit.query | Obtain SKU net content unit information |
| temu.local.goods.illegal.vocabulary.check | Verify whether there are any violations in the product name, product description, and other information |
| bg.local.goods.compliance.rules.getbg.local.goods.compliance.extra.template.get | Used to query product compliance and governance attribute requirements |
| bg.local.goods.compliance.info.fill.list.query | Used to obtain product compliance information and fill in |
| bg.local.goods.add | Create a new product |
