# Endpoints and Request Method

# 

Overview

The Temu Open Platform provides API access based on the HTTPS protocol. Developers (ISVs) who have joined the open platform can follow the steps below to make requests and complete API calls. The following guide is intended for developers implementing API calls independently. 

# 

API Request Methods

We have removed many calling methods for the convenience and security of overall calls, although these methods are more convenient in some scenarios. Currently, we only support the POST method for API interaction, and all request methods are performed using the POST method.
