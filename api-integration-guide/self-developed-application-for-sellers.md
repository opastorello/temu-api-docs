# Self-developed Application for sellers

## 

**Entrance** 

United States: [https://partner-us.temu.com/](https://partner-us.temu.com/) 

Europe: [https://partner-eu.temu.com/](https://partner-eu.temu.com/) 

Other countries/regions: [https://partner.temu.com/](https://partner.temu.com/) 

How to register: Create a Partner Platform account -> Link a Seller Center account-> Fill in app information -> Do the compliance and security questionnaires-> Wait for approval

## 

Step of apply

**Step 1: Create a Partner Platform account **

1. Create a Partner Platform account if you do not already have one.

![image](https://bstatic.kwcdn.com/open-outer/20237f3dc4/81f99c52684f08c6515a6a1ea9c017c2.png)

2.Select a partner type once registration is complete.

![image](https://bstatic.kwcdn.com/open-outer/20237f7164/b5008d3420c92dab206923cdf72ffd22.jpg)

If you have a self-developed system or want to apply for a self-developed app, select **'Self-developed applications seller'.** 

**Notes:**

**- A Partner Platform account only supports one platform type. If more than one partner type applies to you, please create separate Partner Platform accounts.**

**- If you select the wrong partner type, this cannot be changed and you will need to apply for another account. Please select the partner type carefully.**

---

**Step2: Link a Seller Center account**

After completing the steps above, you can select the business type. 

If you are a local seller, select 'You are a local seller'. You will then be redirected to the Seller Center to give authorization.

If you are a cross-border seller, select 'You are a cross-border seller'. You will then be redirected to the Seller Center to give authorization.

![image](https://bstatic.kwcdn.com/open-outer/20237f3dc4/523c408e211d814ade7cad6966c0787e.png)

**Notes:**

**- You are required to link a Seller Center account. **

**- The authorized Seller Center account must be the primary account. **

**- Each Seller Center account can only apply for one self-developed app. **

**- Each Seller Center account can only be linked to one Partner Platform account. If the linked Seller Center account is already linked to another Partner Platform account, linking will be unsuccessful. **

**- Each Partner Platform account can be linked to multiple stores. If you have more than one store, you can manage them from a single Partner Platform account.**

**app name：**you can edit the app name and if you haven't got a Chinese app name you can copy your English app name to it.

1. 

**shopid: **Check that your store's information is correct.

1.

1.

Product website URL: The login URL for the self-developed system
